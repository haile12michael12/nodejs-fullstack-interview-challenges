# Streaming File Uploader - Frontend

This is the frontend for the streaming file uploader application. It provides a user interface for uploading files with real-time progress tracking.

## Features

- Drag and drop file upload
- Real-time upload progress tracking
- File listing and deletion
- Security scan results display

## Installation

```bash
npm install
```

## Running the Application

```bash
npm run dev
```

The application will start on port 5173 by default.
